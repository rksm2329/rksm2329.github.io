C++ STL - 迭代器

C++ 迭代器

迭代器（iterator)，是用来访问 C++ STL 容器中的元素的对象，与指针类似，在使用中可以看作指针。

迭代器主要支持三个运算：++, --, *。++, -- 用来移动迭代器（后继，前驱，要双向迭代器才能访问前驱），* 用来取出迭代器的值。

首尾迭代器 .begin(), .end() 指向首元素和最后一个元素后面的位置。

vector 的迭代器是可以随机访问的，这个迭代器可以跟一个整数加减，vector 的两个迭代器相减得到对应下标的距离。

C++11 可以使用 next(it), prev(it) 来获取迭代器的前驱和后继，此方法主要用于不可以加减的迭代器（非 vector 的迭代器）。

如果迭代器指向的是具有多个成员的对象，可以使⽤ -> 运算符来访问成员。
