C++ 集合

集合（set）是包含不同元素的有序集合容器，使用比较操作，底层红黑树。

头文件：<set>

定义：set<类型，[比较器]> 容器名，可以用初始化列表或者另一个 set 初始化。不指定比较器的时候用 <。

  • s.count(x)， 返回容器内 的数量，
  • s.find(x)，返回容器内指向值为 的元素的迭代器，如果不存在则返回 .end()，

multiset

多重集合，头⽂件、定义、初始化、绝⼤部分操作和成员与 set ⼀致，但有⼏个特殊点。

令 为多重集容器⼤⼩， 为容器中的⼀个元素，multiset<int> s ，令 为 在多重集中出现的次数。

  • s.count(x) 返回 在多重集中的出现次数，
  • s.erase(x)，删除多重集容器内所有值为 的元素，
  • 如果只想查找是否存在元素 、删除⼀个值为 的元素，使⽤ s.find(x) != s.end() 和 auto it = s.find(x); if (it != s.end()) s.erase(it)。直接删除尾迭代器报错。
