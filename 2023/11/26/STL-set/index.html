C++ 集合

集合（set）是包含不同元素的有序集合容器，使用比较操作，底层红黑树。

头文件：<set>

定义：set<类型，[比较器]> 容器名，可以用初始化列表或者另一个 set 初始化。不指定比较器的时候用 <。

对于语句：

时间分别为：

  • $O(1)$
  • $O(M \log M)$
  • $O(M)$

成员函数

定义 $N$ 为容器 s 的大小，$x$ 为元素，$it$ 为迭代器。

修改

  • s.clear() 清空，$O(N)$

  • s.insert(x) 插入元素

    • s.insert(x) $O(\log N)$
    • s.insert({1, 2, 3, 4, ....., M}) $O(M \log (M+N))$

  • s.erase(x) 删除元素，容器不包含元素就不删。

    • s.erase(x) 删除 $x$
    • s.erase(it) 删除 $it$ 指向的元素

  • s.swap(other), swap(s, other) 交换，$O(1)$，仅交换地址

    • swap 对于普通数组时间复杂度 $O(N)$

  • =

    • s = other，用容器赋值，$O(|N| + |other|)$
    • s = {1, 2, 3, ...., M}，用初始化列表赋值，$O(N+M) \log (N+M)$

查找

  • s.count(x)， 返回容器内 $x$ 的数量，$O(\log N)$
  • s.find(x)，返回容器内指向值为 $x$ 的元素的迭代器，如果不存在则返回 .end()，$O(\log N)$
  • s.lower_bound(x) 和 s.upper_boung(x)，返回容器内指向⼤于等于和⼤于 $x$ 的第⼀个元素的迭代器， 如果不存在则返回 s.end()，$O(\log N)$
    • 不是二分，跟 lower_bound(), upper_bound() 区分
    • 直接用 set<int> s; lower_bound(s.begin(), s.end(), x) 时间复杂度线性
  • s.empty(), s.size()
  • s.begin(), s.end()

比较

==, !=, <, <=, >, >=，当 lhs 和 rhs 大小不同时，lhs == rhs, lhs != rhs 时间为 $O(1)$，其他情况使用字典序，$O(N)$

语法结构同 priority_queue，但是不⽤像 priority_queue 反过来写⽐较器。可以使⽤⽐较器函数对象，可以重载运算符。

multiset

多重集合，头⽂件、定义、初始化、绝⼤部分操作和成员与 set ⼀致，但有⼏个特殊点。

令 $N$ 为多重集容器⼤⼩，$x$ 为容器中的⼀个元素，multiset<int> s ，令 $M$ 为 在多重集中出现的次数。

  • s.count(x) 返回 $x$ 在多重集中的出现次数，$O(\log N+M)$
  • s.erase(x)，删除多重集容器内所有值为 $x$ 的元素，$O(\log N+M)$
  • 如果只想查找是否存在元素 、删除⼀个值为 $x$ 的元素，使⽤ s.find(x) != s.end() 和 auto it = s.find(x); if (it != s.end()) s.erase(it)。直接删除尾迭代器报错。
