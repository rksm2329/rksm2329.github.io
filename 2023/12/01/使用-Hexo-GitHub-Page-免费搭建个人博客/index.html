本篇仅对于 Windows。

首先，如果你没有 账号，就去注册一个，网址是 github.com。用户名建议全部小写英文，因为后面要访问 username.github.io，有大写或者中文就很麻烦。

然后，点击右上角新建一个仓库，名称叫 username.github.io，username 为你的用户名。下面什么都不要点，直接点新建。

打开后，把这个标签页留着，不要刷新。

打开 Node.js 网站，两个里面选一个，下载。

然后打开安装包，一路 Next 就可以了。

打开 cmd，输入 node -v 和 npm -v，如果显示了版本，那么安装成功。

然后在安装目录下新建 node_cache 和 node_global 两个文件夹。

安装完之后，打开搜索，搜 环境变量，打开，点击环境变量，然后点开用户变量的 Path 变量，找到 C:\Users\AppData\Roaming\npm，改成安装目录下的 node_global 路径。

然后在系统变量处新建一个变量 NODE_PATH，内容是安装目录+node_global\node_moudles，比如 C:\Program Files\nodejs\node_global\node_moudles，如果 node_global 目录下没有建立 node_moudles 文件夹，那么需要手动建一个。

然后打开系统变量的 Path 变量，添加 %NODE_PATH%。

然后管理员打开 cmd，输入 npm install express -g，如果没有错误，就是成功了。
