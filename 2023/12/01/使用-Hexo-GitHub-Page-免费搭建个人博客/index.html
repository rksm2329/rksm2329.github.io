$$
\large \text{Part 1 Hexo, GitHub 是什么}
$$

$$
\text{Part 1.1 \small Hexo}
$$

$\text{Hexo}$ 是一个 $\text{Node.js}$ 程序，用来生成博客页面，比如 index.html, about.html。

而生成的文件需要放在一个服务器上托管，这个服务器就是 $\text{GitHub}$。

$$
\text{Part 1.2 \small GitHub}
$$
$\text{GitHub}$ 是一个源代码托管网站，这里用作托管 $\text{Hexo}$ 生成的文件。

$$
\large \text{Part 2 配置}
$$

$$
\text{Part 2.1 \small GitHub 相关}
$$

首先，如果你没有 $\text{GitHub}$ 账号，就去注册一个，网址是 github.com。用户名建议全部小写英文，因为后面要访问 username.github.io，有大写或者中文就很麻烦。

然后，点击右上角新建一个仓库，名称叫 username.github.io，username 为你的用户名。下面什么都不要点，直接点新建。

打开后，把这个标签页留着，不要刷新。

$$
\small \text{Part 2.2.2 Node.js}
$$

打开 Node.js 网站，两个里面选一个，下载。

然后打开安装包，一路 Next 就可以了。

打开 cmd，输入 node -v 和 npm -v，如果显示了版本，那么安装成功。

然后在安装目录下新建 node_cache 和 node_global 两个文件夹。

安装完之后，打开搜索，搜 环境变量，打开，点击环境变量，然后点开用户变量的 Path 变量，找到 C:\Users\AppData\Roaming\npm，改成安装目录下的 node_global 路径。

然后在系统变量处新建一个变量 NODE_PATH，内容是安装目录+node_global\node_moudles，比如 C:\Program Files\nodejs\node_global\node_moudles，如果 node_global 目录下没有建立 node_moudles 文件夹，那么需要手动建一个。

然后打开系统变量的 Path 变量，添加 %NODE_PATH%。

然后管理员打开 cmd，输入 npm install express -g，如果没有错误，就是成功了。

$$
\small \text{Part 2.2.3 Watt Toolkit}
$$

如果有了 $\text{GitHub}$ 的加速器，可以跳过此段。

$\text{Watt Toolkit}$，是一个加速器，原本用来加速 $\text{Steam}$，但是也可以加速 $\text{GitHub}$。

Windows 10 及以上可以通过 Microsoft Store 安装。

打开 Watt Toolkit 网站，点击下载，然后打开，安装就行了。
