ABC329C Count xxx 题解

Description

给定一个字符串，求其中有多少个所有字母均相同子串。

Solution

这题一看就知道是最长连号的改版，但是我太菜了，无脑写了个双指针。

所以此题提供两种思路。

双指针

双指针 $O(N)$ 查找子串，设两个指针分别为 $i,j$，开一个标记数组统计一下每个字符的出现次数，然后判断：

  • 如果当前的字符已经出现过了：
    • 如果出现次数小于现在的出现次数（$j-i+1$），那么就更新答案和标记数组。
    • 否则就什么也不干。
  • 否则，就记录次数，更新答案。

正解

开一个 $cnt$ 变量来记录出现次数，$ans$ 数组来统计出现次数，对于每个 $0 \le i \lt N$:

  • 如果 $i=0$ 或 $s_i \ne s_{i-1}$，那么 $cnt$ 变为 $1$；
  • 否则，$cnt$ 加 $1$
  • 然后每次更新 $ans_{s_{i}}$ 的数值即可。
