ABC331E Set Meal 题解

Description

给定 个主菜和 个配菜，有 个套餐不能配在一起，求套餐最大价格。

Solution

这题有一个奇怪的做法：利用题目的特点做。

具体怎么做呢？就是用 map 标记不能匹配的菜，对 数组从大到小排序（注意 数组要记录编号），然后枚举每个 ，也就是枚举 数组，再枚举每个 ，也就是枚举 数组，如果找到第一个没有被标记过的 ，那么退出，更新答案。

为什么能过呢？因为这个算法均摊时间复杂度为 ！

大概结构：

  • 枚举 ：
    • 枚举 ：
      • 如果没有被标记过，退出；
      • 否则，继续。
    • 更新答案。
