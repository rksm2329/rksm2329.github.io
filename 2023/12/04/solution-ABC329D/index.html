ABC329D Election Quick Report 题解

ABC329D Election Quick Report 题解

Description

给定 $N,M$ 和 $M$ 个数，表示有 $N$ 个候选人和 $M$ 张选票，请你求出对于每个 $1 \le i \le N$：当投了前 $i$ 票时，谁的票数最多，如果有票数相同的输出编号最小的。

Solution

是不是感觉有点像 P7072 [CSP-J2020] 直播获奖？这题就是直播获奖的简化版！

于是，我们就可以开一个 multiset 记录每个候选人的票数和编号，然后输入的时候就删除再插入就可以了。为什么要这么做呢？因为 set 不支持修改元素！如果你这么做了，就会告诉你你修改的那一项是只读的。

注意：如果使用 pair，那么不能使用自带比较器。
